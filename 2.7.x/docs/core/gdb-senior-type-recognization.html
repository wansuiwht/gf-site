使用 GoFrame ORM 查询数据时，由于字段值是一个泛型对象，返回的数据类型将会被自动识别映射到 Go 变量类型，作为泛型的具体值。

例如：

  • 当字段类型为 int(xx) 时，查询到的字段值类型将会被识别会 int 类型
  • 当字段类型为 varchar(xxx)/ char(xxx)/ text 等类型时将会被自动识别为 string 类型
  • ……
