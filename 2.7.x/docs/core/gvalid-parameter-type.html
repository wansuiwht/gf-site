📄️ 数据校验-单数据校验

在GoFrame框架中如何进行单数据校验。它讨论了通过Data方法指定被校验数据，以及通过Rule方法指定校验规则的使用方法。实例演示了校验数据长度、数据类型和大小、以及正则表达式校验等不同的校验场景，包括如何应用多个自定义错误提示。
